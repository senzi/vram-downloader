# VRAM Downloader

一个讽刺性的单页应用，模拟硬件零售网站提供"虚拟显存下载"服务。

## Live Demo

👉 [在线体验](https://vram.closeai.moe)

## 简介

这是一个基于 Vue3 + Vite 的网页应用，旨在调侃当今 AI 大模型对显存的巨大需求。通过模拟下载显存的过程，以幽默的方式展示了显存作为硬件资源的特性。

## 特点

- 🎮 提供多种显存容量选择
- 🌈 NVIDIA 风格的界面设计
- 📊 逼真的下载进度显示
- 🎯 自定义显存容量选择
- 😄 富有趣味性的下载体验

## 技术栈

- Vue 3
- Vite
- JavaScript
- CSS3

## 开发

```bash
# 安装依赖
npm install

# 开发服务器
npm run dev

# 构建
npm run build
```
